# Local Operator Docs

This public repo intentionally does **not** commit the detailed Codex/operator workflow guide.

That material is kept locally outside version control for privacy.

## Public Repo Truth

- live app entrypoint: `src/main.jsx` -> `src/App.jsx`
- `src/App.jsx` is the only app root
- sheets are the source of truth for tracked demand and progress
- `breedingSessions` are the source of truth for live execution state
- queue and planner are derived systems
